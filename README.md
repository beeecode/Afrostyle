# Afrostyle — Modern Tailor Management System

Afrostyle is a premium, specialized dashboard designed for African fashion designers and tailors. It streamlines the entire workflow from order booking and measurement management to portfolio showcase and earnings tracking.

![Afrostyle Dashboard](https://images.unsplash.com/photo-1558769132-cb1aea458c5e?auto=format&fit=crop&q=80&w=1200)

## ✨ Core Features

- **📊 Comprehensive Dashboard**: Get a bird's-eye view of your active orders, pending balances, and monthly performance.
- **🧵 Order Management**: Track every stage of the tailoring process — from initial measurement to final delivery.
- **📏 Measurement Vault**: Store and manage detailed body measurements for all your customers for easy reference.
- **💰 Earnings Tracking**: Monitor your income, pending payouts, and transaction history with ease.
- **📸 Portfolio Management**: Showcase your best work and style references in a dedicated portfolio section.
- **📱 Responsive Design**: Fully optimized for both desktop and mobile, so you can manage your shop on the go.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Beecode/Afrostyle.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🛠 Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Components**: Custom UI components and [Radix UI](https://www.radix-ui.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 📝 Project Structure

- `app/`: Next.js App Router pages and layouts.
- `components/`: Reusable UI elements and feature-specific components.
- `public/`: Static assets and images.
- `styles/`: Global CSS and Tailwind configurations.
